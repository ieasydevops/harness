# Subagent: 废弃代码扫描

## 任务

扫描项目源代码和资源文件，识别未使用的类型、函数、变量、文件。逐文件读取并检查，输出废弃代码清单。

## 扫描范围

读取 {{项目绝对路径}}/{{源码目录}}/ 目录下全部源代码文件，以及 {{项目绝对路径}}/{{资源目录}}/ 目录下的资源文件。

## 检查规则

1. 未使用的类型 -- 已定义但未被任何其他文件引用的 struct、class、enum、protocol。
2. 未使用的函数/方法 -- 已定义但未被调用的 func（排除 @objc、override、协议实现等需保留的情况）。
3. 未使用的变量/常量 -- 已定义但未被读取的 let/var（排除框架要求的属性包装器等特殊声明）。
4. 未使用的文件 -- 整个文件中定义的所有公开符号均未被其他文件引用。
5. 无效导入 -- import 的模块在当前文件中未被使用。
6. 过期注释 -- 注释中引用了不存在的类型、函数、文件名。

## 输出格式

```
## 废弃代码 扫描结果

违规数量：N

| # | 文件 | 行号 | 违规项 | 说明 |
|---|------|------|--------|-----|
| 1 | ... | ... | ... | ... |

未发现违规的检查项：...
```
